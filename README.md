# Weather ETL Project

## Описание
Этот проект реализует простой **ETL-пайплайн**, который ежедневно забирает данные о погоде через OpenWeather API и сохраняет их в CSV-файл `weatherData.csv`.  
Проект предназначен для изучения основ **дата-инжиниринга**, включая работу с API, трансформацию данных и хранение истории.

---

## Цели проекта
- Получить практический опыт работы с API
- Научиться извлекать нужные поля из JSON
- Создавать аккуратную историю данных
- Сохранять данные в CSV для последующей аналитики

---

## Стек технологий
- Python (requests, csv, datetime)
- OpenWeather API
- CSV для хранения данных
